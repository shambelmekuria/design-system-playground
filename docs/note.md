# Section Padding
If ALL sections use the same padding:
- Page feels flat
- No visual hierarchy
- Everything looks equally important
` Good UI needs variation with control`

# Use 3-level section system
1. Small sections
    For:
        - testimonials
        - FAQs
        - small blocks
        - py-12 sm:py-16 lg:py-20
 2. Default sections
    For:
        - features
        - services
        - content sections
        - py-16 sm:py-20 lg:py-24
3. Large sections
    For:
        - big CTA blocks
        - important highlights
        - py-20 sm:py-24 lg:py-32
